# Convert HWC to CHW with C++

## Update Submodule

```bash
$ git submodule update --init
```

## Build

```bash
$ chmod +x cmake-build.sh
$ ./cmake-build.sh # include `cp build/cv_example.*.so ./cv_example.so`
```

## Run

```bash
$ python test_cpp_hwc_to_chw.py
```

## Test

```bash
$ python py_hwc_to_chw.py
```
