# Socket

Socket 是一種用來在不同設備或程序之間進行網絡通信的機制。它提供了一個統一的 API，使開發者可以在不同的操作系統上使用 `TCP` 或 `UDP` 來傳輸數據。

Socket 的基本概念:

- `TCP (Transmission Control Protocol)`: 面向連接的協議，提供可靠的數據傳輸（有確認機制）。
- `UDP (User Datagram Protocol)`: 無連接的協議，不保證數據送達，但效率高。
- `IP 地址`: 設備在網絡中的唯一標識，例如 192.168.1.1。
- `端口號 (Port)`: 用來區分同一設備上不同的應用程序，範圍 0-65535。
