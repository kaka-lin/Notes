## Prepare

### 1. Install database

#### Ubuntu

```bash
$ sudo apt-get update && sudo apt-get upgrade
$ sudo apt-get install sqlite3
```

#### Mac

```bash
$ brew install openssl
$ brew install mysql
$ brew install mysql-connector-c
```

### 2. Install requirement packages

```bash
$ pip3 install -r requirements.txt
```
