Python:

- [ ] insertion

## Usage

Example:

```
    4
   / \
  2   5
 / \   \
1   3   6
         \
          7
           \
            8

Preorder: [4, 2, 1, 3, 5, 6, 7, 8]
Inorder: [1, 2, 3, 4, 5, 6, 7, 8]
Postoder: [1, 3, 2, 8, 7, 6, 5, 4]
Level-order: [4, 2, 6, 1, 3, 5, 7]
```

### C/C++

- Build

    ```sh
    $ ./cmake-build.sh
    ```

- Run

    ```sh
    $ cd build
    $ ./bstree
    ```
